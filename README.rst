==============================================================================
Prequisites
==============================================================================

- [OpenBLAS])(https://github.com/xianyi/OpenBLAS)

OpenBLAS installation guide:

::

    $ git clone https://github.com/xianyi/OpenBLAS.git
    $ cd OpenBLAS
    $ mkdir _build
    $ cd _build
    $ cmake ..
    $ make
    $ sudo make install
